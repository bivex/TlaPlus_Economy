================================================================================
ГЛАВА 5. ЭЛАСТИЧНОСТЬ И ЕЁ ПРИМЕНЕНИЕ
Анализ по структуре: ФАКТ — НОРМА — ТРЕБОВАНИЕ
================================================================================

--------------------------------------------------------------------------------
ЭЛАСТИЧНОСТЬ СПРОСА ПО ЦЕНЕ: РАСЧЁТ
--------------------------------------------------------------------------------

[ФАКТ]
- Эластичность спроса по цене = % изменения величины спроса / % изменения цены.
- Пример: 10%-ный рост цены → 20%-ное снижение спроса → эластичность = 2.
- Величина спроса обратно пропорциональна цене → процентные изменения имеют противоположные знаки.
- В книге знак минус опускается; используются абсолютные (положительные) значения.

[НОРМА]
- Большая эластичность подразумевает большую степень реакции величины спроса на изменение цены.

[ТРЕБОВАНИЕ]
- Понимать суть эластичности (сила реакции) важнее, чем формулу расчёта.

--------------------------------------------------------------------------------
МЕТОД СРЕДНЕЙ ТОЧКИ
--------------------------------------------------------------------------------

[ФАКТ]
- Проблема: эластичность из A в B отличается от B в A из-за разной базы процентных изменений.
- Пример: точка A (€4, 120), точка B (€6, 80).
  - Из A в B: цена +50%, количество −33% → эластичность 0,66.
  - Из B в A: цена −33%, количество +50% → эластичность 1,5.
- Метод средней точки: процентное изменение = изменение / среднее значение × 100.
- Пример: средняя цена €5; изменение €4→€6 = (6−4)/5 × 100 = 40%.
- Формула: Эластичность = (Q2−Q1)/[(Q2+Q1)/2] ÷ (P2−P1)/[(P2+P1)/2].

[НОРМА]
- Метод средней точки даёт одинаковый результат независимо от направления изменения.

[ТРЕБОВАНИЕ]
- Использовать метод средней точки при необходимости расчёта эластичности.

--------------------------------------------------------------------------------
МНОГООБРАЗИЕ КРИВЫХ СПРОСА
--------------------------------------------------------------------------------

[ФАКТ]
Пять случаев (рис. 5.1):
1. Абсолютно неэластичный (эластичность = 0): вертикальная кривая; величина спроса не зависит от цены.
2. Неэластичный (эластичность < 1): крутая кривая; % изменения спроса < % изменения цены.
3. Единичная эластичность (эластичность = 1): % изменения спроса = % изменения цены.
4. Эластичный (эластичность > 1): пологая кривая; % изменения спроса > % изменения цены.
5. Абсолютно эластичный (эластичность → ∞): горизонтальная кривая; малейшее изменение цены → огромное изменение спроса.

- Практическое правило (при равных масштабах осей): чем положе кривая → тем больше эластичность; чем круче → тем меньше.
- Мнемоника: неэластичные кривые похожи на букву I; эластичные — на среднюю черту буквы E.

[НОРМА]
- Классификация: >1 — эластичный; <1 — неэластичный; =1 — единичная эластичность.

[ТРЕБОВАНИЕ]
- Различать пять типов эластичности спроса по форме кривой.

--------------------------------------------------------------------------------
ОПРЕДЕЛЯЮЩИЕ ФАКТОРЫ ЭЛАСТИЧНОСТИ СПРОСА ПО ЦЕНЕ
--------------------------------------------------------------------------------

[ФАКТ]
1. Близость товаров-заменителей: больше заменителей → более эластичный спрос.
2. Определение рынка: узкие рынки более эластичны (продукты питания < мороженое < ванильное мороженое).
3. Доля дохода: больше доля дохода → выше эластичность (мебель vs мороженое при 10%-ном росте цены).
4. Временные рамки: долгосрочный период → более эластичный спрос (бензин: краткосрочно — незначительно; долгосрочно — экономичные авто, переезд, общественный транспорт).
   - Электроэнергия vs газ: краткосрочно спрос на электричество неэластичен (уже установлено оборудование); долгосрочно — замена на газовое оборудование.

[НОРМА]
- Узко ограниченные рынки → более эластичный спрос.
- Товары склонны приобретать более эластичный спрос в более длительных периодах.

[ТРЕБОВАНИЕ]
- (Явных требований нет — описательные факторы.)

================================================================================
ОБЩИЕ РАСХОДЫ, ОБЩАЯ ВЫРУЧКА И ЭЛАСТИЧНОСТЬ
================================================================================

[ФАКТ]
- Общие (совокупные) расходы покупателей = P × Q (цена × количество купленного).
- Общая (совокупная) выручка продавцов = P × Q (цена × количество проданного).
- Графически: площадь прямоугольника под кривой спроса.
- Пример (рис. 5.2): P=€4, Q=100 → расходы = €400.

Правила:
1. Спрос неэластичен (<1): цена и общие расходы движутся в одном направлении.
   - Пример (рис. 5.3): цена €1→€3, спрос 100→80, расходы €100→€240 (рост).
2. Спрос эластичен (>1): цена и общие расходы движутся в разных направлениях.
   - Пример (рис. 5.4): цена €4→€5, спрос 50→20, расходы €200→€100 (падение).
3. Единичная эластичность (=1): общие расходы неизменны при изменении цены.

[НОРМА]
- При неэластичном спросе рост цены увеличивает P×Q (уменьшение Q пропорционально меньше роста P).
- При эластичном спросе рост цены уменьшает P×Q (уменьшение Q пропорционально больше роста P).

[ТРЕБОВАНИЕ]
- При анализе рынка оценивать эластичность, чтобы прогнозировать изменение выручки/расходов.

--------------------------------------------------------------------------------
ЭЛАСТИЧНОСТЬ И ОБЩИЕ РАСХОДЫ НА ЛИНЕЙНОЙ КРИВОЙ СПРОСА
--------------------------------------------------------------------------------

[ФАКТ]
- Линейная кривая спроса имеет постоянный наклон, но непостоянную эластичность.
- Наклон = отношение изменений; эластичность = отношение процентных изменений.
- В точках с высокой ценой и малым Q → спрос эластичен.
- В точках с низкой ценой и большим Q → спрос неэластичен.
- Пример (рис. 5.5, таблица):
  - P=€7, Q=0 → эластичность 3,7 (эластичный)
  - P=€4, Q=6 → эластичность 1,0 (единичная)
  - P=€1, Q=12 → эластичность 0,1 (неэластичный)
- Между P=€3 и P=€4 — единичная эластичность; общие расходы одинаковы (€24).

[НОРМА]
- Наклон ≠ эластичность; линейная кривая содержит все три типа эластичности.

[ТРЕБОВАНИЕ]
- Не путать постоянный наклон с постоянной эластичностью.

================================================================================
ДРУГИЕ ВИДЫ ЭЛАСТИЧНОСТИ СПРОСА
================================================================================

--------------------------------------------------------------------------------
ЭЛАСТИЧНОСТЬ СПРОСА ПО ДОХОДУ
--------------------------------------------------------------------------------

[ФАКТ]
- Эластичность спроса по доходу = % изменения величины спроса / % изменения дохода.
- Нормальные товары: положительная эластичность (рост дохода → рост спроса).
- Худшие товары (низшего уровня): отрицательная эластичность (рост дохода → снижение спроса; пример: автобус).
- Товары первой необходимости: небольшая эластичность по доходу (еда, одежда).
- Предметы роскоши: большая эластичность по доходу (икра, бриллианты).

[НОРМА]
- Потребители покупают предметы первой необходимости независимо от уровня доходов; предметы роскоши — только при высоких доходах.

[ТРЕБОВАНИЕ]
- (Явных требований нет.)

--------------------------------------------------------------------------------
ПЕРЕКРЁСТНАЯ ЭЛАСТИЧНОСТЬ СПРОСА ПО ЦЕНЕ
--------------------------------------------------------------------------------

[ФАКТ]
- Перекрёстная эластичность = % изменения спроса на товар 1 / % изменения цены товара 2.
- Взаимозаменяемые товары (субституты): положительная перекрёстная эластичность (рост цены бифштексов → рост спроса на шницели).
- Взаимодополняющие товары (комплементы): отрицательная перекрёстная эластичность (рост цены компьютеров → снижение спроса на ПО).
- Перекрёстная эластичность может увеличиваться с течением времени (электроэнергия → газ: краткосрочно эффект мал, долгосрочно — значителен).

[НОРМА]
- Знак перекрёстной эластичности определяет тип связи между товарами.

[ТРЕБОВАНИЕ]
- (Явных требований нет.)

================================================================================
ПРИВЛЕКАЙТЕ БОЛЬШЕ ПАССАЖИРОВ! (КЕЙС)
================================================================================

[ФАКТ]
- Автобусная компания: стандартная цена €3, вместимость 80 мест.
- Уровень заполнения: 6–8 утра ~95%; 8–16 ~30%; 16–18 ~90%; после 18 ~20%.
- Взаимозаменяемые услуги: личные авто, такси, местная железная дорога.
- Стратегия: утро/вечер +€1 (до €4); дневные рейсы −50% (до €1,50); после 18:30 — прекращение.
- Результаты через 6 месяцев: утро 95%→90%; вечер 90%→87%; дневные 30%→65%.
- Общие расходы пассажиров выросли → общая выручка выросла.

[НОРМА]
- Правильное ценообразование требует понимания ценовой эластичности спроса.
- Увеличение цены может быть выгодным при неэластичном спросе (пиковые часы).

[ТРЕБОВАНИЕ]
- Анализировать эластичность спроса до принятия ценовых решений.

================================================================================
ЭЛАСТИЧНОСТЬ ПРЕДЛОЖЕНИЯ
================================================================================

--------------------------------------------------------------------------------
ЭЛАСТИЧНОСТЬ ПРЕДЛОЖЕНИЯ ПО ЦЕНЕ И ОПРЕДЕЛЯЮЩИЕ ФАКТОРЫ
--------------------------------------------------------------------------------

[ФАКТ]
- Эластичность предложения по цене = % изменения величины предложения / % изменения цены.
- Предложение эластично: величина предложения значительно реагирует на цену.
- Предложение неэластично: лишь слегка реагирует.
- Ключевой фактор — временной промежуток: предложение более эластично в долгосрочном периоде.
- Примеры: прибрежная земля (неэластично — невозможно произвести больше); книги/авто/телевизоры (эластично — можно менять масштаб производства).
- В краткосрочном периоде: невозможно быстро построить новые производства; в долгосрочном — можно.

[НОРМА]
- Предложение обычно обладает большей эластичностью в долгосрочном периоде.

[ТРЕБОВАНИЕ]
- (Явных требований нет.)

--------------------------------------------------------------------------------
МНОГООБРАЗИЕ КРИВЫХ ПРЕДЛОЖЕНИЯ (РИС. 5.6)
--------------------------------------------------------------------------------

[ФАКТ]
Пять случаев:
1. Абсолютно неэластичное (эластичность = 0): вертикальная кривая.
2. Неэластичное (<1): крутая кривая.
3. Единичная эластичность (=1).
4. Эластичное (>1): пологая кривая.
5. Абсолютно эластичное (→ ∞): горизонтальная кривая.

- Пример расчёта: молоко, цена €2,85→€3,15 (+10%), объём 90 000→110 000 (+20%) → эластичность = 2.

[НОРМА]
- Аналогично спросу: положе кривая → выше эластичность.

[ТРЕБОВАНИЕ]
- (Явных требований нет.)

--------------------------------------------------------------------------------
ИЗМЕНЕНИЕ ЭЛАСТИЧНОСТИ ВДОЛЬ КРИВОЙ ПРЕДЛОЖЕНИЯ (РИС. 5.7)
--------------------------------------------------------------------------------

[ФАКТ]
- При малых объёмах: неиспользуемые мощности → эластичность высокая.
  - Пример: цена €3→€4 (+29%), предложение 100→200 (+67%) → эластичность >1.
- При больших объёмах: полная загрузка → эластичность низкая.
  - Пример: цена €12→€15 (+22%), предложение 500→525 (+5%) → эластичность <1.

[НОРМА]
- Компании с неиспользуемыми мощностями могут значительно реагировать на цену; при полной загрузке — реакция минимальна без значительного увеличения цены.

[ТРЕБОВАНИЕ]
- Понимать, что эластичность предложения может меняться вдоль одной кривой.

--------------------------------------------------------------------------------
ОБЩАЯ ВЫРУЧКА И ЭЛАСТИЧНОСТЬ ПРЕДЛОЖЕНИЯ
--------------------------------------------------------------------------------

[ФАКТ]
- Принципы общих расходов/выручки спроса применимы и к предложению.
- Выручка для продавцов = расходы для покупателей (P × Q).

[ТРЕБОВАНИЕ]
- Помнить, что выручка продавцов = расходы покупателей.

================================================================================
ТРИ ПРИМЕНЕНИЯ ПРЕДЛОЖЕНИЯ, СПРОСА И ЭЛАСТИЧНОСТИ
================================================================================

--------------------------------------------------------------------------------
МОГУТ ЛИ ХОРОШИЕ НОВОСТИ ДЛЯ ОТРАСЛИ КОМПЬЮТЕРОВ
ОКАЗАТЬСЯ ПЛОХИМИ ДЛЯ ПРОИЗВОДИТЕЛЕЙ ЧИПОВ?
--------------------------------------------------------------------------------

[ФАКТ]
- Новый материал для чипов → рост производительности → кривая предложения сдвигается вправо (S1→S2).
- Кривая спроса не меняется.
- Результат: цена чипов падает (€10→€4), количество растёт (100→110).
- Спрос на чипы неэластичен → общая выручка падает (€1000→€440).
- Реальный пример: графений; DRAM: $55/МБ в начале 1990-х → $1 в начале 2000-х.
- Производители, первыми начавшие выпуск, получили огромную прибыль; по мере входа новых фирм предложение росло, цены падали, ряд фирм вышли из бизнеса.

[НОРМА]
- То, что плохо для производителей, не обязательно плохо для общества: потребители платят меньше за большие мощности.
- Каждый производитель является ценополучателем и рационально использует新技术, но когда все делают то же самое — положение ухудшается.

[ТРЕБОВАНИЕ]
- Анализировать последствия технологических изменений с учётом эластичности.

--------------------------------------------------------------------------------
ПОЧЕМУ ЦЕНЫ НА ГОРНОЛЫЖНЫХ КУРОРТАХ СИЛЬНО РАЗЛИЧАЮТСЯ?
--------------------------------------------------------------------------------

[ФАКТ]
- ~1,3 млн жителей Великобритании посетили европейские горнолыжные курорты в 2010 г.
- Пример цен (табл. 5.1, Австрия, 7 дней, выезд из Лондона):
  - 5 декабря: €470; 26 декабря: €1000; 15 января: €650; 20 февраля: €1180.
- Предложение ограничено (конечное число оборудованных мест) → неэластично в краткосрочном периоде.
- В пиковые периоды (каникулы): спрос резко растёт → цены значительно увеличиваются.
- В непиковые периоды: избыточные мощности → предложение более эластично → рост цен умеренный.
- Аналогичные рынки: авиа- и ж/д перевозки, электроэнергия — ценовая дискриминация по времени.

[НОРМА]
- Ограниченность предложения при неэластичном спросе в пиковые периоды → значительный рост цен.
- Фирмы осуществляют ценовую дискриминацию потребителей.

[ТРЕБОВАНИЕ]
- Понимать, что ограниченность предложения + неэластичный спрос = значительные колебания цен.

--------------------------------------------------------------------------------
ПРИВОДИТ ЛИ ЗАПРЕТ НАРКОТИКОВ К УВЕЛИЧЕНИЮ ИЛИ УМЕНЬШЕНИЮ ПРЕСТУПЛЕНИЙ?
--------------------------------------------------------------------------------

[ФАКТ]
- Запрет наркотиков → борьба с контрабандой → снижение предложения → кривая S сдвигается влево.
- Спрос не меняется.
- Результат: цена растёт (P1→P2), количество падает (Q1→Q2) → потребление снижается.
- Но: спрос на наркотики неэластичен → повышение цены увеличивает общую выручку на рынке наркотиков → наркозависимые нуждаются в ещё больших суммах → количество преступлений может вырасти.
- Альтернатива: разъяснительная работа → снижение спроса (D сдвигается влево) → и цена, и количество падают → общая выручка падает → количество преступлений снижается.
- Долгосрочная перспектива: спрос на наркотики может быть более эластичен (высокие цены предотвращают эксперименты молодёжи) → запрет может снизить преступления в долгосрочном периоде.

[НОРМА]
- Запрет наркотиков снижает потребление, но может увеличить преступления (краткосрочно).
- Разъяснительная работа снижает и потребление, и преступления.

[ТРЕБОВАНИЕ]
- Анализировать политику с учётом эластичности и временных рамок.

СНИЖЕНИЕ ЦЕН В КОНСОЛЬНЫХ ВОЙНАХ (КЕЙС)
================================================================================

[ФАКТ]
- Лето 2009 г.: Sony снизила цену PS3 с ~€300 до €299 (Европа), £249,99 (Великобритания), $299 (США).
- Конкуренты: Xbox 360 — от €129, Wii — ~€169.

- Sony рассчитывала на высокую краткосрочную эластичность спроса по цене.

[НОРМА]
- При ценовых решениях фирмы учитывают эластичность, реакцию конкурентов, взаимозаменяемость и взаимодополняемость товаров (игры для консолей).

[ТРЕБОВАНИЕ]
- Рассматривать ценовые решения в контексте эластичности, конкуренции и взаимосвязанных товаров.

ИТОГИ
================================================================================

ВЫВОДЫ:

[ФАКТ]
- Эластичность спроса по цене: >1 — эластичный; <1 — неэластичный; =1 — единичная.
- Факторы эластичности спроса: заменители, тип товара, границы рынка, доля дохода, время.
- Неэластичный спрос: цена и выручка движутся в одном направлении.
- Эластичный спрос: цена и выручка движутся в противоположных направлениях.
- Эластичность спроса по доходу: нормальные товары (+), худшие товары (−).
- Перекрёстная эластичность: субституты (+), комплементы (−).
- Эластичность предложения по цене: зависит от временного промежутка; долгосрочно — более эластична.

[НОРМА]

[ТРЕБОВАНИЕ]
