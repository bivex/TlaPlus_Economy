ГЛАВА 13. МОНОПОЛИЯ
Анализ по структуре: ФАКТ — НОРМА — ТРЕБОВАНИЕ

--------------------------------------------------------------------------------
ВВЕДЕНИЕ: ПРИЧИНЫ ВОЗНИКНОВЕНИЯ МОНОПОЛИЙ
--------------------------------------------------------------------------------

[ФАКТ]
- Пример: Microsoft Windows — ~90% рынка операционных систем; ЕС оштрафовала Microsoft на $1,4 млрд.
- Монополия — фирма, являющаяся единственным продавцом на рынке.
- Три причины возникновения монополий:
  1. Монопольная собственность на ключевые ресурсы (пример: алмазная компания De Beers).
  2. Правительственное создание монополий (патенты, авторские права — дают фирме исключительное право на производство продукта).
  3. Естественная монополия — единственная фирма может предлагать товар целому рынку с меньшими издержками, чем множество фирм (пример: водоснабжение; высокая FC, низкие MC → убывающая ATC).
- Предпринимательский рост: фирмы могут расти через слияния и поглощения, усиливая рыночную власть.

[НОРМА]
- Патенты и авторские права создают монополии, но одновременно стимулируют инновации и творчество.
- Естественная монополия — следствие экономии от масштаба, а не злоумышленных действий.

[ТРЕБОВАНИЕ]
- Понимать три причины возникновения монополий и уметь приводить примеры каждой.

КРИВАЯ СПРОСА МОНОПОЛИИ
================================================================================

--------------------------------------------------------------------------------
СПРОС МОНОПОЛИСТА vs КОНКУРЕНТНОЙ ФИРМЫ
--------------------------------------------------------------------------------

[ФАКТ]
- Рис. 13.3: конкурентная фирма — горизонтальная кривая спроса; монополия — нисходящая кривая спроса.
- Конкурентная фирма — ценополучатель: может продать любое количество по рыночной цене.
- Монополия — единственный продавец: кривая спроса фирмы совпадает с рыночной кривой спроса.
- Монополист может выбрать любую точку на кривой спроса, но не вне её.
- Чтобы продать больше, монополия обязана снижать цену.

[НОРМА]
- Цель монополиста — максимизация прибыли (как и конкурентной фирмы).

[ТРЕБОВАНИЕ]
- Понимать различие кривых спроса для конкурентной и монополистической фирм.

--------------------------------------------------------------------------------
ВЫРУЧКА МОНОПОЛИИ
--------------------------------------------------------------------------------

[ФАКТ]
- Табл. 13.1: пример с водоснабжением (Q от 0 до 8, P от €1,00 до €0,30).
- TR = P × Q; AR = TR / Q = P; MR = ΔTR / ΔQ.
- Ключевой результат: MR монополиста всегда меньше цены (MR < P).
- Два эффекта при увеличении объёма:
  + Эффект объёма: Q растёт → TR увеличивается.
  + Эффект цены: P снижается → TR уменьшается.
- У конкурентной фирмы эффект цены отсутствует → MR = P.
- Рис. 13.4: кривая MR лежит ниже кривой спроса; MR может быть отрицательным (когда эффект цены > эффекта объёма).

[НОРМА]
- Для конкурентной фирмы: MR = P.
- Для монополии: MR < P (из-за необходимости снижать цену для увеличения продаж).

[ТРЕБОВАНИЕ]
- Понимать, почему маржинальная выручка монополии ниже цены товара.
- Уметь объяснять два эффекта (объёма и цены), влияющих на выручку монополии.

================================================================================
МАКСИМИЗАЦИЯ ПРИБЫЛИ МОНОПОЛИИ
================================================================================

--------------------------------------------------------------------------------
ПРАВИЛО MR = MC
--------------------------------------------------------------------------------

[ФАКТ]
- Рис. 13.5: кривые спроса, MR и MC монополии.
- При Q1 (низкий выпуск): MR > MC → увеличение Q повышает прибыль.
- При Q2 (высокий выпуск): MR < MC → уменьшение Q повышает прибыль.
- Qmax: MR = MC → максимизирующий прибыль объём (точка А).
- Затем монополист использует кривую спроса для определения цены (точка В).
- Для конкурентной фирмы: P = MR = MC.
- Для монополии: P > MR = MC.
- Ключевое различие: на конкурентном рынке P = MC; на монопольном P > MC.

[НОРМА]
- Правило максимизации прибыли (MR = MC) едино для конкурентной фирмы и монополии.
- Различие — в соотношении цены и MR: у конкурента P = MR; у монополиста P > MR.

[ТРЕБОВАНИЕ]
- Уметь определять максимизирующий прибыль объём и цену монополии на графике.

--------------------------------------------------------------------------------
ПОЧЕМУ У МОНОПОЛИИ НЕТ КРИВОЙ ПРЕДЛОЖЕНИЯ
--------------------------------------------------------------------------------

[ФАКТ]
- Монополия не имеет кривой предложения.
- Кривая предложения показывает количество при любой данной цене — это имеет смысл только для ценополучателей.
- Монополист сам устанавливает цену одновременно с определением объёма.
- Решение монополиста о предложении невозможно отделить от формы кривой спроса.

[НОРМА]
- Решения о предложении конкурентных фирм можно анализировать без знания кривой спроса; для монополии — нельзя.

[ТРЕБОВАНИЕ]
- Понимать, почему монополия не имеет кривой предложения.

--------------------------------------------------------------------------------
ПРИБЫЛЬ МОНОПОЛИИ
--------------------------------------------------------------------------------

[ФАКТ]
- Прибыль = TR − TC = (P − ATC) × Q.
- Рис. 13.6: прибыль = площадь прямоугольника BCDE (высота = P − ATC, основание = Q).
- Монополист может получать положительную экономическую прибыль даже в долгосрочном периоде (в отличие от конкурентной фирмы).

[НОРМА]
- Монопольная прибыль может существовать в долгосрочном периоде, так как вход на рынок заблокирован.

[ТРЕБОВАНИЕ]
- Уметь измерять прибыль монополии на графике.

--------------------------------------------------------------------------------
ЛЕКАРСТВЕННЫЕ ПРЕПАРАТЫ: МОНОПОЛИЯ vs КОНКУРЕНЦИЯ (ПРИМЕР)
--------------------------------------------------------------------------------

[ФАКТ]
- Рис. 13.7: рынок типичного лекарства (MC = const).
- В период патента: монополия → цена выше MC, объём ниже эффективного.
- После истечения патента: вход дженериков → конкурентный рынок → цена = MC.
- Реальность подтверждает теорию: цены дженериков ниже монопольных.
- Некоторые потребители остаются лояльными бренду → бывший монополист сохраняет небольшую надбавку.

[НОРМА]
- Патентная защита создаёт временную монополию, стимулируя инновации, но по истечении патента конкуренция снижает цену.

[ТРЕБОВАНИЕ]
- (Явных требований нет — описательный пример.)

ОБЩЕСТВЕННЫЕ ИЗДЕРЖКИ МОНОПОЛИИ
================================================================================

--------------------------------------------------------------------------------
БЕЗВОЗВРАТНЫЕ ПОТЕРИ
--------------------------------------------------------------------------------

[ФАКТ]
- Критерий эффективности: совокупный излишек = потребительский излишек + излишек производителя.
- Рис. 13.8: эффективный объём — точка пересечения кривых спроса и MC (где ценность для потребителей = MC).
- Благожелательный общественный плановик установил бы P = MC.
- Рис. 13.9: монополия производит меньше эффективного объёма.
- Безвозвратные потери (DWL) — треугольник между кривой спроса и кривой MC.
- Монопольное ценообразование препятствует взаимовыгодным сделкам: потребители, оценивающие товар выше MC, но ниже монопольной цены, не покупают.
- Монополист подобен частному сборщику налогов: «вбивает клин» между готовностью платить и издержками, как и налог (гл. 8).
- Разница: налоговые поступления получает правительство; монопольную прибыль — частная фирма.

[НОРМА]
- Монопольный рынок не обеспечивает эффективного распределения ресурсов (совокупный излишек не максимизируется).

[ТРЕБОВАНИЕ]
- Уметь показывать безвозвратные потери монополии на графике.
- Понимать аналогию между монопольным ценообразованием и налогообложением.

--------------------------------------------------------------------------------
МОНОПОЛЬНАЯ ПРИБЫЛЬ: ИЗДЕРЖКИ ДЛЯ ОБЩЕСТВА?
--------------------------------------------------------------------------------

[ФАКТ]
- Монопольная прибыль сама по себе не уменьшает размер «экономического пирога» — она перераспределяет его от потребителей к монополисту.
- Проблема монополии — не прибыль, а неэффективно низкий объём производства (безвозвратные потери).
- Исключение: если монополия несёт дополнительные издержки для сохранения своего положения (лоббисты, юридическая поддержка), то общественные издержки включают и эти расходы.

[НОРМА]
- С точки зрения экономической эффективности, проблема монополии — в безвозвратных потерях, а не в перераспределении прибыли.

[ТРЕБОВАНИЕ]
- Различать перераспределительный эффект монопольной прибыли и безвозвратные потери.

--------------------------------------------------------------------------------
ИСТОРИЯ О ЦЕНЕ НА КНИГУ
--------------------------------------------------------------------------------

[ФАКТ]
- Ценовая дискриминация — продажа одного и того же товара разным покупателям по различным ценам при одинаковых издержках.
- На конкурентном рынке ценовая дискриминация невозможна.
- Пример: издательство, 100 тыс. фанатов (готовы платить €30) + 400 тыс. обычных читателей (готовы платить €5).
- Единая цена €30: TR = €3 млн, прибыль = €1 млн. Единая цена €5: TR = €2,5 млн, прибыль = €0,5 млн.
- Ценовая дискриминация (Швейцария €30 + Турция €5): TR = €5 млн, прибыль = €3 млн.
- Безвозвратные потери при единой цене: 400 тыс. читателей × €5 = €2 млн.

[НОРМА]
- (Явных норм нет — описательный пример.)

[ТРЕБОВАНИЕ]

--------------------------------------------------------------------------------
ТРИ УРОКА ЦЕНОВОЙ ДИСКРИМИНАЦИИ
--------------------------------------------------------------------------------

[ФАКТ]
- Вывод 1: ценовая дискриминация — стратегия максимизации прибыли. Монополист назначает каждому покупателю цену, близкую к его готовности заплатить.
- Вывод 2: ценовая дискриминация требует разделения покупателей по готовности платить (география, возраст, доход, время суток).
- Арбитражирование — покупка на дешёвом рынке и перепродажа на дорогом — препятствует ценовой дискриминации.
- Вывод 3: ценовая дискриминация может увеличить экономическое благосостояние (устраняет безвозвратные потери), но весь прирост достаётся производителю.

[НОРМА]
- При совершенной ценовой дискриминации безвозвратные потери устраняются, но потребительский излишек = 0.

[ТРЕБОВАНИЕ]
- Знать три вывода о ценовой дискриминации.

--------------------------------------------------------------------------------
АНАЛИЗ ЦЕНОВОЙ ДИСКРИМИНАЦИИ
--------------------------------------------------------------------------------

[ФАКТ]
- Совершенная ценовая дискриминация: монополист точно знает готовность каждого покупателя платить и назначает индивидуальные цены.
- Рис. 13.10 (а): единая цена → безвозвратные потери + потребительский излишек > 0.
- Рис. 13.10 (б): совершенная ценовая дискриминация → DWL = 0, потребительский излишек = 0, вся выгода — прибыль монополиста.
- В реальности совершенная ценовая дискриминация недостижима; фирмы делят покупателей на группы.
- Несовершенная ценовая дискриминация может увеличить, уменьшить или не изменить совокупный излишек (однозначного ответа нет).
- Однозначный вывод: ценовая дискриминация увеличивает прибыль монополии.

[НОРМА]
- Совершенная ценовая дискриминация устраняет безвозвратные потери, но перераспределяет весь излишек от потребителей к монополисту.

[ТРЕБОВАНИЕ]
- Различать совершенную и несовершенную ценовую дискриминацию.

ПРИМЕРЫ ЦЕНОВОЙ ДИСКРИМИНАЦИИ
--------------------------------------------------------------------------------

[ФАКТ]
- Билеты в кино: скидки для детей и пожилых (меньшая готовность платить).
- Авиабилеты: низкие цены при ночёвке с субботы на воскресенье — разделение бизнес-пассажиров (высокая готовность платить) и туристов (низкая).
- Купоны на скидку: разделение по готовности тратить время (обеспеченные занятые люди vs безработные).
- Оптовые скидки: цена за единицу снижается с объёмом покупки; готовность платить за каждую дополнительную единицу убывает.

ЗАКОНОДАТЕЛЬСТВО О КОНКУРЕНЦИИ (АНТИТРЕСТОВСКОЕ)
--------------------------------------------------------------------------------

[ФАКТ]
- Четыре способа реагирования государства на монополии:
  1. Превращение монополизированных отраслей в конкурентные (антитрестовское законодательство).
  2. Регулирование поведения монополий.
  3. Государственная собственность (национализация).
  4. Бездействие.
- США: Закон Шермана (1890 г.) — первый антитрестовский закон.
- Европа: национальные органы по конкуренции + Комиссия по конкуренции ЕС + Европейская сеть по развитию конкуренции.
- Великобритания: Комиссия по конкуренции; Закон о конкуренции 1998 г.; Закон о предприятиях 2002 г.
- Законодательство о конкуренции обеспечивает:
  + Борьбу с картелями и ограничительной практикой.
  + Запрет фиксации цен, хищнического ценообразования.
  + Мониторинг слияний и поглощений.
- Синергизм — выгоды от объединения компаний (снижение издержек через эффективность).

[НОРМА]
- Правительство должно оценивать: выгоды от синергизма vs издержки от снижения конкуренции.

[ТРЕБОВАНИЕ]
- Знать четыре способа государственной политики в отношении монополий.

РЕГУЛИРОВАНИЕ
--------------------------------------------------------------------------------

[ФАКТ]
- Регулирование — обычная практика для естественных монополий (водоснабжение, газ, электроэнергия).
- Ценообразование по MC: цена = MC → эффективное распределение ресурсов, но монополист несёт убытки (MC < ATC при убывающей ATC).
- Рис. 13.11: при цене = MC → убытки, фирма покидает отрасль.
- Два способа решения:
  1. Субсидирование монополиста (налоги → безвозвратные потери).
  2. Цена = ATC → нулевая прибыль, но DWL (цена > MC).
- Проблема качества регулирования: монополист лишён стимулов к снижению издержек.
- Великобритания: ценовое регулирование по верхнему пределу (RPI − X, например инфляция минус 2%).
- Если производительность растёт быстрее X → прибыль увеличивается.

[НОРМА]
- Ценообразование по MC эффективно, но порождает убытки естественной монополии.
- Ценообразование по ATC устраняет убытки, но создаёт безвозвратные потери.

[ТРЕБОВАНИЕ]
- Понимать две проблемы регулирования естественной монополии по принципу P = MC.

--------------------------------------------------------------------------------
ВЫВОДЫ
--------------------------------------------------------------------------------

[ФАКТ]
- Монополия — единственный продавец; возникает через владение ресурсом, правительственные права или естественную монополию.
- Спрос монополии — нисходящая кривая; MR < P.
- Максимизация прибыли: MR = MC → P > MC (в отличие от конкуренции: P = MC).
- Монополия производит меньше эффективного объёма → безвозвратные потери.
- Политические реакции: антитрестовское законодательство, регулирование, национализация, бездействие.
