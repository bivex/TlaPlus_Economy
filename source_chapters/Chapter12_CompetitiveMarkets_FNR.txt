================================================================================
ГЛАВА 12. ФИРМЫ НА КОНКУРЕНТНЫХ РЫНКАХ
Анализ по структуре: ФАКТ — НОРМА — ТРЕБОВАНИЕ
================================================================================

--------------------------------------------------------------------------------
ВВЕДЕНИЕ
--------------------------------------------------------------------------------

[ФАКТ]
- Пример: автозаправочная станция (повышение цены на 20% → резкое сокращение продаж) vs водопроводная компания (монополия → снижение потребления незначительно).
- Конкурентный рынок: множество покупателей и продавцов; каждый очень мал по сравнению с размером рынка → незначительная возможность влиять на рыночные цены.
- Власть над рынком: фирма способна оказывать серьёзное воздействие на цену.
- Цель главы — исследовать процесс принятия фирмой решения об объёмах производства на конкурентных рынках; пролить свет на кривую предложения.

[НОРМА]
- Различие в структуре рынков отражается на решениях о формировании цен и объёмах производства.

[ТРЕБОВАНИЕ]
- Понимать, какие виды издержек (постоянные, переменные, маржинальные) наиболее существенны для принятия решения о величине предложения.

================================================================================
ЧТО ТАКОЕ КОНКУРЕНТНЫЙ РЫНОК?
================================================================================

--------------------------------------------------------------------------------
ПОНЯТИЕ КОНКУРЕЦИИ
--------------------------------------------------------------------------------

[ФАКТ]
- Конкуренция: наличие конкурентов → потребители имеют выбор (качество, цена, обслуживание).
- Характеристики конкуренции:
  - Более одной фирмы с одинаковыми/похожими продуктами.
  - Взаимозаменяемые товары (газ vs электроэнергия).
  - Чем выше взаимозаменяемость, тем больше конкуренция.
  - Фирмы влияют на конкуренцию через отношения с потребителями.
- Совершенно конкурентный рынок — два основных свойства:
  1. Множество покупателей и продавцов.
  2. Предлагаемая продукция — однородная.
- Следствие: действия отдельного продавца/покупателя имеют незначительное влияние на рыночную цену → ценополучатели.
- Два дополнительных свойства:
  3. Свободный вход на рынок и выход с него.
  4. Высокая степень осведомлённости о рынке.
- Пример: рынок молока — ни один покупатель/продавец не может повлиять на цену.
- Технологии: интернет, блоги, обзоры повышают прозрачность цен и качества.

[НОРМА]
- Свободный вход и выход — мощные силы, влияющие на результаты работы конкурентных рынков в долгосрочном периоде.

[ТРЕБОВАНИЕ]
- Знать характеристики конкурентного рынка и понимать, почему фирмы являются ценополучателями.

--------------------------------------------------------------------------------
ВЫРУЧКА КОНКУРЕНТНОЙ ФИРМЫ
--------------------------------------------------------------------------------

[ФАКТ]
- Пример: семейная молочная ферма Гранди. Цена €6/литр.
- Общая выручка TR = P × Q. При Q = 1000: TR = €6000.
- Цена не зависит от объёма выпуска (ценополучатель) → TR прямо пропорциональна Q.
- Табл. 12.1: при любом Q цена = €6; TR = 6Q; AR = 6; MR = 6.
- Средняя выручка (AR) = TR/Q. Для любой фирмы AR = P.
- Маржинальная выручка (MR) = изменение TR при продаже дополнительной единицы.
- Для конкурентной фирмы: MR = P (цена — константа).

[НОРМА]
- Общее правило: для любой фирмы средняя выручка равна цене товара.
- Правило конкурентной фирмы: маржинальная выручка равна цене товара.

[ТРЕБОВАНИЕ]
- Понимать, почему для конкурентной фирмы AR = MR = P.

================================================================================
МАКСИМИЗАЦИЯ ПРИБЫЛИ И КРИВАЯ ПРЕДЛОЖЕНИЯ
================================================================================

--------------------------------------------------------------------------------
ПРОСТОЙ ПРИМЕР МАКСИМИЗАЦИИ ПРИБЫЛИ
--------------------------------------------------------------------------------

[ФАКТ]
- Табл. 12.2: ферма Гранди, FC = €3, P = €6.
- Прибыль максимальна при Q = 4 или Q = 5 (прибыль = €7).
- Маржинальный принцип:
  - 1-й литр: MR = €6, MC = €2 → прибыль растёт на €4.
  - 2-й: MR = €6, MC = €3 → прибыль растёт на €3.
  - 5-й: MR = €6, MC = €5 → прибыль не меняется.
  - 6-й: MR = €6, MC = €7 → прибыль падает на €1.
- Пока MR > MC → увеличение прибыли; MR < MC → уменьшение прибыли.

[НОРМА]
- Рациональные люди мыслят в терминах маржинальных изменений (Принцип 1).
- Фирма естественным образом приходит к максимизирующему прибыль количеству продукции через маржинальные приращения.

[ТРЕБОВАНИЕ]
- Понимать маржинальный подход к максимизации прибыли: производить, пока MR ≥ MC.

--------------------------------------------------------------------------------
КРИВАЯ МАРЖИНАЛЬНЫХ ИЗДЕРЖЕК И РЕШЕНИЕ О ВЕЛИЧИНЕ ПРЕДЛОЖЕНИЯ
--------------------------------------------------------------------------------

[ФАКТ]
- Рис. 12.1: кривые MC (наклон вверх), ATC (U-образная), горизонтальная линия P = MR = AR.
- При Q1: MR > MC → увеличение Q повышает прибыль.
- При Q2: MR < MC → уменьшение Q повышает прибыль.
- Qmax: MR = MC → максимизирующий прибыль объём.
- Общее правило: при максимизирующем прибыль объёме MR = MC.
- Для конкурентной фирмы: P = MC → объём предложения определяется точкой пересечения цены и MC.
- Рис. 12.2: при повышении цены с P1 до P2 фирма увеличивает производство с Q1 до Q2.
- Кривая MC конкурентной фирмы одновременно является кривой предложения.

[НОРМА]
- Кривая маржинальных издержек определяет количество товара, которое фирма желает предложить при любой цене → она является кривой предложения.

[ТРЕБОВАНИЕ]
- Понимать, что кривая предложения конкурентной фирмы = часть кривой MC.

--------------------------------------------------------------------------------
РЕШЕНИЕ ФИРМЫ О ПРЕКРАЩЕНИИ ПРОИЗВОДСТВА В КРАТКОСРОЧНОМ ПЕРИОДЕ
--------------------------------------------------------------------------------

[ФАКТ]
- Временная приостановка: прекращение выпуска в краткосрочном периоде; FC сохраняются.
- Выход с рынка: покидание рынка в долгосрочном периоде; FC не несутся.
- Критерий временного прекращения:
  - TR < VC → прекратить производство.
  - Или: P < AVC (цена меньше средних переменных издержек).
- Фирма приостанавливает производство, если цена не покрывает средних переменных издержек.
- Кривая предложения в краткосрочном периоде: часть кривой MC, лежащая выше AVC (рис. 12.3).

[НОРМА]
- При временном прекращении производства фирма сравнивает цену со средними переменными (не общими!) издержками.

[ТРЕБОВАНИЕ]
- Знать критерий: фирма временно прекращает производство, если P < AVC.

--------------------------------------------------------------------------------
БЕЗВОЗВРАТНЫЕ ИЗДЕРЖКИ
--------------------------------------------------------------------------------

[ФАКТ]
- Безвозвратные издержки — издержки, которые уже понесены и не могут быть возмещены.
- Противоположность альтернативных издержек: альтернативные — связаны с выбором; безвозвратные — невозможно избежать независимо от выбора.
- Безвозвратные издержки следует игнорировать при принятии решений.
- FC — безвозвратные в краткосрочном периоде (фирма не может их вернуть).
- Пример: потерянный билет в кино за €7. Выгода от фильма = €10 > альтернативные издержки нового билета €7 → следует купить новый. Утерянные €7 — безвозвратные.
- "Не жалей о пролитом молоке."

[НОРМА]
- Безвозвратные издержки нерелевантны для принятия решений.

[ТРЕБОВАНИЕ]
- При принятии решений игнорировать безвозвратные издержки.

--------------------------------------------------------------------------------
ПОЛУПУСТЫЕ РЕСТОРАНЫ И МИНИ-ГОЛЬФ (ПРИМЕР)
--------------------------------------------------------------------------------

[ФАКТ]
- Ресторан: FC (аренда, оборудование) — безвозвратные в краткосрочном периоде. Владелец учитывает только VC (продукты, труд) при решении об открытии в обеденное время.
- Мини-гольф: FC (аренда земли и сооружений) — нерелевантны. Открывать только когда выручка > VC.

[НОРМА]
- При решении о продолжении/приостановке деятельности учитывать только переменные издержки.

[ТРЕБОВАНИЕ]
- (Явных требований нет — практические примеры.)

--------------------------------------------------------------------------------
РЕШЕНИЕ ФИРМЫ О ВХОДЕ НА РЫНОК ИЛИ ВЫХОДЕ В ДОЛГОСРОЧНОМ ПЕРИОДЕ
--------------------------------------------------------------------------------

[ФАКТ]
- Критерий выхода с рынка: TR < TC, или P < ATC → фирма покидает рынок.
- Критерий входа на рынок: P > ATC → фирма входит.
- Кривая предложения в долгосрочном периоде: часть кривой MC, лежащая выше ATC (рис. 12.4).
- Прибыль = (P − ATC) × Q.
  - P > ATC → прибыль > 0.
  - P < ATC → убыток.
  - P = ATC → нулевая прибыль.

[НОРМА]
- (Явных норм нет — формальные критерии.)

[ТРЕБОВАНИЕ]
- Различать краткосрочный критерий (P < AVC → приостановка) и долгосрочный (P < ATC → выход).

--------------------------------------------------------------------------------
ИЗМЕРЕНИЕ ПРИБЫЛИ НА ГРАФИКЕ
--------------------------------------------------------------------------------

[ФАКТ]
- Рис. 12.5 (а): фирма с прибылью — площадь прямоугольника (P − ATC) × Q.
- Рис. 12.5 (б): фирма с убытком — площадь прямоугольника (ATC − P) × Q.
- Прибыль = (P − ATC) × Q (высота × основание прямоугольника).

[НОРМА]
- (Явных норм нет — графический метод.)

[ТРЕБОВАНИЕ]
- Уметь измерять прибыль и убыток на графике конкурентной фирмы.

--------------------------------------------------------------------------------
ВЫХОД ИЗ БИЗНЕСА В КРИЗИС (КЕЙС — 2008–2009)
--------------------------------------------------------------------------------

[ФАКТ]
- Глобальный спад 2008–2009: снижение объёмов продаж, нехватка кредитных ресурсов.
- Мелкие фирмы: конкурентные рынки, ограниченное влияние на цену → часто работают на грани прибыльности → снижение спроса → убытки → закрытие.
- 2008 г. (Великобритания): банкротства выросли на 30% → 23 879 ликвидированных компаний.
- Крупные фирмы: приостановка производства в краткосрочном периоде.
  - Honda (Великобритания): остановка на 4 месяца.
  - Aston Martin, Vauxhall, Mini — временные остановки.
  - Fiat: остановка на 2 недели (январь 2010).
  - General Motors: полное прекращение завода в Антверпене.
- Продажи автомобилей в Западной Европе: снижение на 60% от пика 2007 г. (до 2,5 млн).
- Исландия: −88%; Латвия: −77%; Ирландия: −66%; Испания: −41,6%; Италия: −32,6%.
- IHS Global Insight: ~950 потерянных человеко-дней на компанию в Q1 2009; потеряно ~¼ квартального объёма производства Европы.
- Временная приостановка: сохранение персонала (навыки — инвестиции); издержки найма после восстановления могут превысить экономию от сокращения.

[НОРМА]
- Здания и оборудование — безвозвратные издержки в краткосрочном периоде; навыки работников — инвестиции, которые следует сохранять.

[ТРЕБОВАНИЕ]
- (Явных требований нет — описательный кейс.)

КРИВАЯ ПРЕДЛОЖЕНИЯ КОНКУРЕНТНОГО РЫНКА В ЦЕЛОМ
================================================================================

--------------------------------------------------------------------------------
КРАТКОСРОЧНЫЙ ПЕРИОД: ПОСТОЯННОЕ КОЛИЧЕСТВО ФИРМ
--------------------------------------------------------------------------------

[ФАКТ]
- 1000 одинаковых фирм: каждая производит объём, при котором MC = P.
- Рыночная величина предложения = сумма величин предложения всех фирм.
- Рис. 12.6: рыночное предложение = 1000 × предложение одной фирмы.

[НОРМА]
- (Явных норм нет — агрегирование.)

[ТРЕБОВАНИЕ]
- (Явных требований нет — описание метода.)

--------------------------------------------------------------------------------
ДОЛГОСРОЧНЫЙ ПЕРИОД: ВОЗМОЖНОСТЬ ВХОДА И ВЫХОДА
--------------------------------------------------------------------------------

[ФАКТ]
- Все фирмы имеют одинаковые кривые издержек (допущение).
- Прибыль > 0 → вход новых фирм → рост предложения → снижение цены → прибыль → 0.
- Убытки → выход фирм → снижение предложения → рост цены → прибыль → 0.
- Итог: нулевая экономическая прибыль в долгосрочном равновесии.
- Условие: P = MC = min ATC (эффективный масштаб).
- Рис. 12.7 (а): фирма в долгосрочном равновесии — P = MC = min ATC, прибыль = 0.
- Рис. 12.7 (б): кривая предложения рынка — горизонтальная линия при P = min ATC.
- Количество фирм меняется для удовлетворения спроса при данной цене.
- Нормальная прибыль — минимальная сумма, необходимая для продолжения использования факторов производства.
- Сверхприбыль — прибыль, превышающая нормальную.
- Пример: фермер инвестировал €1 млн (альтернативный процентный доход €50 000/год) + мог зарабатывать €30 000/год на другой работе → альтернативные издержки = €80 000/год. При нулевой экономической прибыли бухгалтерская прибыль = €80 000 (достаточно для удержания в бизнесе).

[НОРМА]
- В долгосрочном периоде на конкурентном рынке со свободным входом и выходом прибыль стремится к нулю.
- Бухгалтерская прибыль при нулевой экономической прибыли является положительной величиной (включает неявные издержки).

[ТРЕБОВАНИЕ]
- Понимать, что нулевая экономическая прибыль не означает отсутствие дохода у владельца.

--------------------------------------------------------------------------------
ИЗМЕНЕНИЕ СПРОСА В КРАТКОСРОЧНОМ И ДОЛГОСРОЧНОМ ПЕРИОДАХ
--------------------------------------------------------------------------------

[ФАКТ]
- Рис. 12.8: рынок молока, молоко обладает целительными свойствами → рост спроса.
- Исходное равновесие (а): точка А, цена P1, объём Q1, нулевая прибыль.
- Краткосрочная реакция (б): спрос D1 → D2 → точка В, цена P2 > P1, объём Q2 > Q1, прибыль > 0.
- Долгосрочная реакция (в): прибыль привлекает новые фирмы → S1 → S2 → точка С, цена возвращается к P1, объём Q3 > Q2 > Q1. Количество фирм увеличилось.

[НОРМА]
- В краткосрочном периоде рост спроса увеличивает цену и приносит прибыль; в долгосрочном — вход новых фирм возвращает цену к min ATC.

[ТРЕБОВАНИЕ]
- Различать краткосрочную и долгосрочную реакцию конкурентного рынка на изменение спроса.

--------------------------------------------------------------------------------
ПОЧЕМУ КРИВАЯ ПРЕДЛОЖЕНИЯ В ДОЛГОСРОЧНОМ ПЕРИОДЕ МОЖЕТ ИМЕТЬ ПОЛОЖИТЕЛЬНЫЙ НАКЛОН
--------------------------------------------------------------------------------

[ФАКТ]
- Две причины положительного наклона:
  1. Некоторые ресурсы доступны в ограниченном количестве (земля → рост цены земли при увеличении числа фермеров → рост издержек).
  2. Разные фирмы имеют разные издержки (рынок малярных услуг: новые входящие имеют более высокие издержки → более высокая цена для привлечения).
- Маржинальная фирма — компания с наивысшими издержками, получающая нулевую прибыль. Фирмы с меньшими издержками получают положительную прибыль даже в долгосрочном периоде.
- Кривая предложения в долгосрочном периоде обычно более эластична, чем в краткосрочном.

[НОРМА]
- Несмотря на причины положительного наклона, основной вывод остаётся верным: кривая предложения в долгосрочном периоде более эластична, чем в краткосрочном.

[ТРЕБОВАНИЕ]
- Понимать причины, по которым долгосрочная кривая предложения может иметь положительный наклон.

================================================================================
ЗАКЛЮЧЕНИЕ
================================================================================

--------------------------------------------------------------------------------
ВЫВОДЫ
--------------------------------------------------------------------------------

[ФАКТ]
- Конкурентная фирма — ценополучатель: AR = MR = P.
- Максимизация прибыли: фирма выбирает объём, при котором MR = MC → P = MC → кривая MC = кривая предложения.
- Краткосрочный критерий прекращения: P < AVC.
- Долгосрочный критерий выхода: P < ATC.
- При свободном входе и выходе прибыль → 0 в долгосрочном равновесии: P = min ATC = MC; все фирмы работают с эффективным масштабом.
- Изменения спроса: краткосрочно — рост цены и прибыли; долгосрочно — рост числа фирм, возврат цены к min ATC.

[НОРМА]
- Маржинальный анализ — ключ к пониманию поведения конкурентной фирмы.
- Цена конкурентного товара близка к издержкам производства (P = MC).
- При свободном входе/выходе цена = минимально возможным средним общим издержкам.
- Результат работы монополистического рынка часто не соответствует интересам общества (предварительный вывод для следующей главы).

[ТРЕБОВАНИЕ]
- Применять маржинальный анализ (MR = MC) для определения оптимального объёма производства конкурентной фирмы.
- Различать критерии прекращения производства в краткосрочном (P < AVC) и долгосрочном (P < ATC) периодах.
- Понимать механизм движения к нулевой экономической прибыли в долгосрочном периоде.
